◆ GetPlotTemplate()

Plot SciRooPlot::PlotManager::GetPlotTemplate ( const string &  plotTemplateName = "1d",
double_t  screenResolution = 100 
)
static

Defines a set of standard plot templates with quadratic axis frame and consistent layout among them.

The following pre-defined layouts are available: 1d, 2d, 1d_ratio and 1d_3panels. By specifying a screen resolution in dpi, the size of the plot displayed in interactive mode (or when saved as bitmap) can be modified. For multiple of 72dpi the resulting plot will have the best agreement with the pdf version.
